# Unit 1 Prime Factorization

This folder contains the first app-ready content scaffold derived from the
current repository docs.

Everything here should remain aligned with:

- `docs/mastery-rules.md`
- `docs/prerequisite-map.md`
- `docs/recommendation-rules.md`
- `docs/suji-subskills-prime-factorization.md`
- `docs/suji-error-patterns-prime-factorization.md`

This content is intentionally conservative. It records skill definitions and
examples, but it does not declare numeric mastery thresholds.

`U1-S5` remains the documented parent skill from `docs/mastery-rules.md`.
Because the repo's planning docs already separate several transfer patterns,
the app scaffold introduces draft child skills under `U1-S5` for finer
diagnosis. Those child skills are app-level scaffolding, not approved mastery
targets yet.
